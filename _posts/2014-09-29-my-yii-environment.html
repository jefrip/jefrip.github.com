Saya baru dalam hal mengembangkan aplikasi web dengan Yii Framework, tetapi sudah menemukan tools dasar untuk pengembangan aplikasi web. Berikut tools yang saya gunakan, diantaranya :

Yii 1.1

Yii PHP Framework (yii-1.1.15.022a51)

- Versi terakhir dari Yii Framework 1.1

Yii Booster 4.0.1 (Menggunakan Twitter Bootstrap 3.1.1 dan min butuh PHP 5.4) atau versi lama Yii Booster 3.1.1 (Menggunakan Twitter Bootstrap 2.3.2 kompatibel dengan PHP 5.3)

- Untuk keperluan template, Form, dan (the awesome) Gii versi bootstrap. (btw saya fans Twitter Bootstrap :D)

Yii-User

- Untuk keperluan manajemen user/login

Rights

- Untuk manajemen role atau hak akses terhadap suatu controller atau action.

Yii 2
